# Пет проект сайта пиццерии с возможностью оформления заказа
В этом проекте был оформлен каталог продуктов с сортировкой по категориям. Реализована регистрация пользователей с e-mail верификацией аккаунта, аутентификация пользователей (в том числе при помощи GitHub), и редактирование профиля. Написана корзина заказов по принципу CRUD и возможность оформления заказа, доступная аутентифицированным пользователям. Изначально использовалась база данных SQLite3, в последствии заменена на PostgreSQL. Фронтенд написан на чистом HTML и CSS
